# TaakLine Logistics Intelligence Platform

This project analyzes logistics shipment data from TaakLine using Python, SQL, and Power BI. The goal was to clean raw order data, perform business analysis, and build an interactive dashboard that provides insights into revenue, customers, and sales trends.

---

## Overview

This project demonstrates a complete analytics workflow, starting with  shipment data and ending with an interactive Power BI dashboard.

The project includes:

- Data cleaning with Python
- SQL queries for business analysis
- Power BI dashboard development
- Version control with Git and GitHub
